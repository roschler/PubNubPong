Welcome to Bricks 3D!

This game uses the following technologies:

WebGL:
A JavaScript API for rendering interactive 3D and 2D graphics within the browser, without the user of any plug-ins, and takes advantage of hardware acceleration on the GPU.


Three.js:
This lightweight JavaScript library has been used for creating and displaying 3D graphics on the web browser using WebGL.


Stats.js:
This class has been used for monitoring the performance of the game.
